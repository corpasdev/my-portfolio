# My Web Portfolio with Astro

> My personal portfolio built with Astro (Latest) + TailwindCSS 4 + Vite.

## Installation

```bash
npm create astro@latest
cd my-portfolio
npm install
npm run dev
```

# Project structure
public/

  pages/

  layouts/

README.md

# Integrations
 * Tailwind CSS (v4 + plugin Vite)
 * Sitemap (@astrojs/sitemap)
 * Adapter Node (@astrojs/node)

# Deploy

```bash
npm run build
npm run preview
```
